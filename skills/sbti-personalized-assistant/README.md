# SBTI Personalized Assistant

基于 SBTI 人格测试的个性化 AI 助手技能，通过分析用户聊天记录自动适配沟通风格。

## 🎯 功能特性

- **人格分析**：读取用户近期聊天记录，分析言行风格
- **SBTI 测试**：模拟完成 32 道人格测试题
- **风格适配**：根据人格结果自动调整 AI 助手的沟通风格
- **27 种人格**：覆盖稀有型、独立型、积极型、谨慎型、幽默型等多种类型
- **结果海报**：生成个性化人格结果海报

## 🚀 快速开始

### 触发方式

在飞书聊天中发送以下任意指令：

- `分析我的 SBTI 人格`
- `测一下我是什么人格`
- `分析人格并适配 AI 风格`
- `生成专属 AI 助手`

### 执行流程

1. **启动**：记录入口埋点，提示用户开始分析
2. **读取消息**：获取用户近期 14-30 天的聊天记录
3. **生成画像**：分析沟通风格、情绪基调、关系模式等
4. **模拟答题**：基于画像生成 32 题答案
5. **计算结果**：运行人格计算脚本
6. **输出报告**：展示人格结果和海报
7. **风格切换**：加载对应人格的 AI 沟通风格

## 📋 用户指令

| 指令 | 功能 |
|------|------|
| `分析我的 SBTI 人格` | 完整流程：分析 → 生成报告 → 适配风格 |
| `重新分析` | 重新读取聊天记录分析 |
| `显示我的配置` | 查看当前 soul.md 内容 |
| `切换成 XX 风格` | 手动切换到指定人格风格 |
| `恢复默认` | 恢复标准 AI 风格 |
| `工作模式` | 切换到高效风格（CTRL） |
| `闲聊模式` | 切换到轻松风格（HHHH） |

## 🔧 依赖说明

- **Python 3.x**：用于运行计算脚本
- **Node.js**：用于飞书 IM 消息读取
- **飞书 API**：需要相应的消息读取权限

## 📝 注意事项

1. **隐私保护**：聊天记录仅用于分析，不存储、不传输
2. **风格平滑**：切换风格时有过渡期，不会突然完全改变语气
3. **用户选择权**：随时可以关闭或修改自适应风格
4. **安全建议**：不因迎合用户而给出错误或危险的建议
