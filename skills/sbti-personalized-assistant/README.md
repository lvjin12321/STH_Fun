## 🎯 这是什么？

**SBTI 个性化助手** 是一个基于 SBTI 人格测试的 AI 技能，通过分析你的聊天记录自动适配沟通风格：

- ✅ **自动分析**：读取你的聊天记录，分析言行风格
- ✅ **人格测试**：模拟完成 32 道 SBTI 人格测试题
- ✅ **风格适配**：根据人格结果自动调整 AI 的沟通方式
- ✅ **27 种人格**：覆盖稀有型、独立型、积极型、谨慎型、幽默型等
- ✅ **结果海报**：生成个性化人格结果海报

**核心理念：** 不是让你适应 AI，而是让 AI 适应你。

## 🚀 快速开始

### 前置要求

- **OpenClaw** 或兼容的 AI Agent 框架
- **Python 3.8+** (用于人格计算脚本)
- **消息读取权限** (用于分析聊天记录)

### 安装步骤

#### 1. 克隆或下载仓库

```bash
git clone https://github.com/lvjin12321/STH_Fun.git
cd STH_Fun/skills/sbti-personalized-assistant
```

#### 2. 安装依赖

```bash
pip install pyyaml requests
```

#### 3. 部署到 OpenClaw

将 `skills/sbti-personalized-assistant` 目录放入 OpenClaw 的 `skills` 文件夹：

```bash
# 假设 OpenClaw 安装在 ~/.openclaw
cp -r skills/sbti-personalized-assistant ~/.openclaw/skills/
```

#### 4. 重启 OpenClaw

```bash
openclaw restart
```

## 💬 使用方式

### 对你的 Agent 说

将本技能部署到 OpenClaw 后，直接在你的聊天窗口中输入以下任意指令：

```
分析我的 SBTI 人格
测一下我是什么人格
分析人格并适配 AI 风格
生成专属 AI 助手
```

### 执行流程

```
1. 你发送指令 → 2. Agent 读取你的聊天记录 → 3. 生成行为画像
       ↓
4. 模拟完成 32 道测试题 → 5. 计算人格结果 → 6. 展示报告 + 海报
       ↓
7. 自动加载对应人格的沟通风格 → 8. 后续对话按新风格回应
```

### 示例

**你：** `分析我的 SBTI 人格`

**Agent：** 
```
🔍 正在分析你的沟通风格...
📊 已读取最近 23 天的聊天记录
📝 生成行为画像中...
✅ 分析完成！你的 SBTI 人格是：CTRL - 拿捏者

🎯 人格特质：目标导向、高效执行、结果驱动
🤖 AI 角色：高效执行助手

正在加载 CTRL 风格... 以后我会用更直接、高效的方式和你沟通！
```

## ⚠️ 注意事项

1. **隐私保护**：聊天记录仅用于分析，不存储、不传输到外部
2. **风格平滑**：切换风格时有过渡期，不会突然完全改变语气
3. **用户选择权**：随时可以关闭或修改自适应风格
4. **安全建议**：AI 不会因迎合用户而给出错误或危险的建议
